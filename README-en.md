# FileSharer

## Features

Share a folder on your computer, and access it from your phone or other devices by entering the IP address and port number in a browser.
If the file is a video, it can be streamed directly online, utilizing the local network speed without consuming data.
It can be used to share study materials to a tablet.
Free up phone space and enjoy watching videos.

## Usage

Download the `dist` folder and run `FileSharer.exe` by double-clicking it. Then, access the shared folder by entering the URL in a browser on another device.

## Troubleshooting

If the video cannot be played, try using a different browser.
